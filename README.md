# ModuGame
모두의 게임의 일부 미니게임을 리메이크(REMAKE) C++/Winapi

## Controls

- 알록달록 색종이
  -  ↑,↓,←, → : Move
- 아슬아슬 비행기
  - 마우스

## Requirements

링커 -> 입력 -> 추가 종속
..\Debug\Lib.lib;
msimg32.lib;
C/C++ -> 일반 -> 추가포함
..\lib
